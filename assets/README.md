# Background Photo

Place a real background photo here and name it:

```text
ptsd-scene.jpg
```

The page already references `assets/ptsd-scene.jpg`. If the image is missing,
the interface uses a soft fallback background so the prototype still runs.

You can also add calm background audio here:

```text
calm-ambient.mp3
hobby-audio.mp3
```

The sound control starts at `Sound On`. Browsers may wait for the first tap
before allowing audio, so the page will try again after the user's first
interaction.
